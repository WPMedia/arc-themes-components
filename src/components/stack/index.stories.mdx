# Stack

The Stack component is analogous to the HTML Flex element, but provides additional functionality
such as adding a divider between elements.
It allows for a both a column or row layout. In addition,
the Stack component allows for alignment of the items in the layout
axis as well over-riding the default gap (also known by the term "gutter") size.

## Usage

```
import { Stack } from '@wpmedia/arc-themes-components';


<Stack>
    <element>Child Element</element>
</Stack>
```

# Paragraph

Outputs a paragraph of text. Allows for plain text as well as text with HTML markup. Provides for truncation as well.

## Usage

```
import { Paragraph } from '@wpmedia/arc-themes-components';


<Paragraph>Text to output</Paragraph>
```

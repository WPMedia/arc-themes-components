# Join

The `Join` component is used to inject a separator between children elements.

## Usage

```jsx
import { Join } from "@wpmedia/arc-themes-components";

const Separator = () => <> • </>;

const List = () => (
	<Join separator={Separator}>
		<>List Item 1</>
		<>List Item 2</>
	</Join>
);
```

# Grid

The `<Grid>` component is an wrapper to the CSS Grid layout in order for Themes to
be able to control two-dimensional grid layout's within blocks.

## Usage

```jsx
import { Grid } from "@wpmedia/arc-themes-components";

const Feature = () => (
	<Grid>
		<Image src="/camera.jpeg" alt="A camera with photos in front of it" />
		<Image src="/canyon.jpeg" alt="Landscape view of a canyon" />
		<Image src="/coffee.jpeg" alt="A cup of coffee" />
	</Grid>
);
```

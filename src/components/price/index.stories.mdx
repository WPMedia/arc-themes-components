# Price

The `Price` component is used to denote a price List and/or Sale price of a given item. The component exposed two sub components which are the only children elements that can be used.

## Usage

```jsx
import { Price } from "@wpmedia/arc-themes-components";

const Feature = () => (
	<Price>
		<Price.Sale>$50</Price.Sale>
		<Price.List>$100</Price.List>
	</Price>
);
```

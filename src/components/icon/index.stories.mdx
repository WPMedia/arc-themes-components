# Icon

Icons are used to provide additional meaning via a visual means in place where
a text label doesn't fit or work for design. The Icon component has the ability
to be rendered in the context of an Image if needed. By default the Icon is
presentatioanl and should be not be used in this context as a standalone item. If
using an Icon as a action (Button) ensure accessibility considerations are addressed.

## Usage

```jsx
import { Icon } from "@wpmedia/arc-themes-components";

const Feature = () => <Icon name="User" />;
```

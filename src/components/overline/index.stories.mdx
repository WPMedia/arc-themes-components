# Overline

By default, the Overline component will render an overline span for use above headlines or headers. If an href is passed as a property, the Overline will instead render as a simple hyperlink for internal navigation. This is not intended for use for external navigation. If you desire this functionality, please utilize the &lt;Link /&gt; component as a wrap to the Overline.

## Usage

```
import { Overline } from '@wpmedia/arc-themes-components';


<Overline>Component Children</Overline>
```
